# Skill Swap Platform

A full-stack web application where users can exchange skills.

## Features
- User registration & login
- Skill matching system
- Request & response system
- Backend APIs using Node.js

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express
- Database: MongoDB

## How to Run

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start
